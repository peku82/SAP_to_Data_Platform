2. LO QUE NECESITAMOS
---------------------
Habilitar al usuario SAP **FI002** las siguientes autorizaciones de
SOLO LECTURA (nunca escritura, nunca SQL de modificación):

Ruta en SAP B1:
  Administración
  → Sistema de inicialización
  → Autorizaciones
  → Autorizaciones generales

Categoría "Herramientas → Consultas":
  ✅  Consultas nuevas                → Autorización COMPLETA
  ✅  Generador de consultas          → Autorización COMPLETA
  ✅  Administrador de consultas      → Autorización COMPLETA
  ✅  Asistente de consulta           → Autorización COMPLETA
  ✅  Consultas sistema               → Autorización COMPLETA (lectura)
  ✅  Consultas usuario               → Autorización COMPLETA (lectura)

NOTA IMPORTANTE: el "Generador de Consultas" de SAP B1 está limitado
a **SELECT** por diseño — no permite INSERT / UPDATE / DELETE / DROP.
Es read-only por la arquitectura del producto. Por eso esta
autorización NO expone ningún riesgo de escritura.

Referencia SAP oficial:
  "The Query Generator is a SAP Business One tool that allows users
   to create their own queries. It is limited to SELECT statements only."
