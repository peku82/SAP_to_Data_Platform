docs/
=====

Documentación del proyecto.

  architecture.txt  — arquitectura técnica, flujo de datos y decisiones.
                      Incluye qué debe habilitar TI para extracción
                      programática.
  usage_guide.txt   — guía paso a paso de uso por fase.
  schema_sap.md     — diccionario de tablas SAP B1 extraídas y sus campos
                      clave (se genera automáticamente tras Fase 1).
  runbook.md        — procedimientos operativos: cómo correr deltas, cómo
                      restaurar backup, cómo rotar credenciales.

Audiencia:
  - Equipo de TI JKK Pack
  - Administrador SAP (consultor externo)
  - Equipo de datos / analítica
